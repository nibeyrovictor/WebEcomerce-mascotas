## 📁 Estructura del Proyecto

La estructura del repositorio está organizada por secciones, cada una con su propio archivo `index.html`, y una carpeta de estilos CSS compartida:

# Estructura de archivos para la tienda de mascotas

```
petshop/
│
├── assets/
│   │   ├── style.css
│   │   ├── home.css
│   │   ├── catalog.css
│   │   └── contact.css
│   │
│   └── images/
│
├── home/
│   └── index.html
│
├── catalog/
│   └── index.html
│
├── contact/
│   └── index.html
│
└── index.html (página principal que redirige a home/index.html)
```

## 🚀 Cómo ejecutar el proyecto

1. **Clonar el repositorio**:
   ```bash
   git clone https://github.com/nibeyrovictor/WebEcomerce-mascotas.git

2. **Acceder a la carpeta del proyecto:**:
   ```bash
   cd WebEcomerce-mascotas

3. **Acceder a carpeta de repositorio y abrir HTML en el navegador:**

   - Haz click sobre `index.html` y verás el home para luego redirigirte a cada sección. 
   - O usar la extensión **Live Server** desde Visual Studio Code para mayor comodidad.
